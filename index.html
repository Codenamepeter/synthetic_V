평가 지표 체계

합성 이미지 묶음의 품질을 평가하는 18개 지표. 자체 운영 지표 12개(유사성·유용성·안전성 × 4)와 TTA 시험 지표 6개 모두 본 도구가 자체 계산한다. 일부 지표는 추가 입력(시드 데이터, VTT 평가, mAP 값)이 필요하며, 없을 경우 N/A 처리된다.

판정 기준 — 자체 12지표는 각 70점 이상이면 PASS. TTA 6지표는 항목별 목표 기준 준용.

자체 운영 지표

3축 × 4지표 = 12개
유사성 (Similarity) 묶음 내 분포 동질성
ID 지표명 산식 / 해석 설명 PASS 기준
S-01 해상도 일관성
score = 100 × (1 / n_unique_resolutions)
고유 해상도가 1종이면 100점, 2종이면 50점, n종이면 100/n점
 단일 해상도로 통일된 묶음일수록 학습 안정성 높음 ≥ 70점
S-02 색상 분포 일관성
score = max(0, 100 − CV(RGB_mean) × 200)   where CV = σ / μ
RGB 평균값의 변동계수(표준편차/평균)가 작을수록 만점. CV 0.5에서 0점
 RGB 채널별 평균값의 변동계수(CV). 묶음 색감의 통일성 ≥ 70점
S-03 밝기 분포 일관성
score = max(0, 100 − CV(luminance) × 200)   where CV = σ / μ
휘도 변동계수가 작을수록 만점. CV 0.5에서 0점
 묶음 내 밝기의 균질성. 조명 조건 통일도 측정 ≥ 70점
S-04 히스토그램 동질성
score = max(0, 100 − avg(χ²) / 100)
χ²(H₁,H₂) = ½ · Σ (H₁ − H₂)² / (H₁ + H₂)
이미지 쌍의 히스토그램 χ² 거리(symmetric) 평균이 작을수록 만점
 색상 히스토그램의 페어와이즈 대칭 Chi-square 거리 ≥ 70점
유용성 (Utility) 학습 데이터로서의 가치
ID 지표명 산식 / 해석 설명 PASS 기준
U-01 묶음 규모
score = min(100, N_images / 5)
이미지 500장이면 만점. 350장이면 70점
 학습 데이터의 절대 수량. 적정 학습 규모 확보 여부 ≥ 70점
U-02 시각 다양성
score = min(100, avg(Hamming(pHash)) × 5)
이미지 쌍의 pHash Hamming 거리 평균이 20 이상이면 만점, 14이면 70점
 perceptual hash의 Hamming 거리 평균. 거리 5 이하는 중복 의심 ≥ 70점
U-03 조명·환경 다양성
score = min(100, σ(luminance) × 2.5)
휘도 표준편차가 40이면 만점, 28이면 70점
 묶음 전체 휘도 표준편차. 다양한 조명 조건 포함도 ≥ 70점
U-04 메타 시나리오 풍부성
score = min(100, n_scenario_fields × 20)
시나리오 필드 5개 이상이면 만점, 4개이면 80점
 metadata.json 내 시간대·기상·각도 등 시나리오 필드 수 ≥ 70점
안전성 (Safety) 재식별·중복 위험 관리
ID 지표명 산식 / 해석 설명 PASS 기준
A-01 중복 위험
score = max(0, 100 − N_dup_pairs / N_pairs × 1000)
중복 의심 쌍(Hamming 거리 ≤ 5)의 비율이 3% 이상이면 0점
 전체 페어 중 거의 동일 이미지 쌍의 비율 (pHash 표준 임계값) ≥ 70점
A-02 클러스터 집중
score = min(100, avg(min_neighbor_dist) × 10)
평균 최근접 이웃 거리가 10 이상이면 만점, 7이면 70점
 각 이미지의 최근접 이웃 평균 거리. 특정 영역 집중도 ≥ 70점
A-03 메타 식별자 노출
score = max(0, 100 − n_sensitive_fields × 30)
민감 필드 1개당 30점 감점. 1개에서 70점, 4개 이상이면 0점
 metadata 내 GPS·이름·ID 등 재식별 가능 필드 검출 ≥ 70점
A-04 파일명 패턴 안전
score = 100 − n_suspicious_patterns × 25
의심 패턴 1개당 25점 감점. 1개에서 75점, 2개에서 50점
 UUID·날짜·연속번호 등 식별자 의심 패턴 검출 ≥ 70점

TTA 시험 지표

6개 항목 (모두 자체 계산 가능 · 일부는 추가 입력 필요)
ID 시험항목 산식 / 해석 설명 PASS 기준
TC1 다양성
평균정확도 = 100 × ΣΣx_ij / (n×m×5)
평가자 응답 점수의 합을 (응답자수 × 문항수 × 5점) 으로 나눠 백분율
 프롬프트 4조건(객체·해상도·시간대·기상) 충족도 ≥ 60%
(5점 ≥ 20%)
TC2 구문정확성
정확도 = (1 − 오류건수 / 전체속성수) × 100
전체 속성 중 오류가 없는 비율을 백분율로 환산
 라벨·메타데이터 파일의 스키마(JSONL) 구문규칙 준수율 ≥ 99.9%
TC3 의미정확성
정확도 = (1 − 오류데이터수 / 전체데이터수) × 100
전체 데이터 중 객체-캡션 불일치가 없는 비율을 백분율로 환산
 이미지 캡션과 실제 객체의 의미적 일치도 ≥ 95%
TC4 유효성 (FID)
FID = ‖μ_X − μ_Y‖² + Tr(Σ_X + Σ_Y − 2(Σ_X·Σ_Y)1/2)
두 분포의 평균 벡터 차이와 공분산 행렬 차이를 합산. 작을수록 유사 (행렬 제곱근 사용)
 실제(시드)와 합성 간 Fréchet Inception Distance 평균 ≤ 50
TC5 유효성 (VTT)
A = (TP + TN) / 전체 × 100
맞춘 개수 / 전체 개수의 백분율. 50%에 가까울수록 좋음
 Visual Turing Test. 50% 근방 = 사람이 구분 못함 40% ~ 60%
TC6 유효성 (성능 유사도)
성능유사도 = |mAP_실제 − mAP_합성|
두 모델의 평균 정밀도(mAP) 차이의 절대값. 작을수록 동등
 실제 학습 모델 vs 합성 학습 모델의 mAP 차이 ≤ 10%
